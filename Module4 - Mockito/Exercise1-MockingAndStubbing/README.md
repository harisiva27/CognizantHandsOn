# Exercise 1 - Mocking and Stubbing

## Objective

Learn how to use Mockito to create mock objects and stub method responses for unit testing.

## Scenario

A service depends on an external API. Instead of calling the real API during testing, Mockito is used to create a mock object and return predefined values.

## Concepts Covered

- Mockito
- Mock Objects
- Stubbing using `when()` and `thenReturn()`
- JUnit 5
- Unit Testing
