
# Module 4 - Mockito

## Overview

This module demonstrates the fundamentals of Mockito for unit testing Java applications. Mockito is used to create mock objects, stub method responses, and verify interactions between objects without relying on external dependencies.

## Exercises Completed

### ✅ Exercise 1 - Mocking and Stubbing

**Objective**
- Create mock objects using Mockito.
- Stub method responses using `when()` and `thenReturn()`.
- Test a service that depends on an external API.

**Files**
- ExternalApi.java
- MyService.java
- MyServiceTest.java

---

### ✅ Exercise 2 - Verifying Interactions

**Objective**
- Verify that methods are invoked on mock objects.
- Test service behavior using Mockito's `verify()` method.

**Files**
- ExternalApi.java
- MyService.java
- VerifyInteractionTest.java
