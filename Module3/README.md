# Module 3 - PL/SQL

This repository contains the mandatory hands-on exercises completed for Module 3 of the Cognizant Java FSE Deep Skilling Program.

## Hands-On Exercises

### Exercise 1 - Control Structures
Implemented PL/SQL control structures using different real-world scenarios. The programs demonstrate the use of loops, conditional statements, and database operations.

### Exercise 2 - Stored Procedures
Implemented PL/SQL stored procedures using different scenarios. The procedures perform database operations such as updating records and displaying results using cursors.

## Concepts Covered

- PL/SQL Blocks
- IF...ELSE
- FOR LOOP
- CURSOR
- Stored Procedures
- UPDATE Statement
- DBMS_OUTPUT
