
# Module 6 - Spring Data JPA

This module introduces the fundamentals of Object-Relational Mapping (ORM), Java Persistence API (JPA), Hibernate, and Spring Data JPA. It demonstrates how Spring Boot simplifies database operations using Spring Data JPA and explains the relationship between JPA, Hibernate, and Spring Data JPA.

## Mandatory Hands-ons Completed

### Exercise 1: Spring Data JPA - Quick Example

- Created a Spring Boot project using Maven.
- Configured the H2 in-memory database.
- Created an `Employee` entity using JPA annotations.
- Implemented `EmployeeRepository` using `JpaRepository`.
- Performed basic database operations using:
  - `save()`
  - `findById()`
- Verified the application through successful console output.

### Exercise 2: Difference between JPA, Hibernate and Spring Data JPA

- Studied the concepts of ORM, JPA, Hibernate, and Spring Data JPA.
- Compared JPA, Hibernate, and Spring Data JPA.
- Understood the relationship between these technologies.
- Learned the advantages of using Spring Data JPA in Spring Boot applications.
