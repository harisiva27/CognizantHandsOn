# Exercise 1 - Configuring a Basic Spring Application

## Objective

Configure a basic Spring application using Spring Core and XML-based configuration.

## Scenario

A Library Management application is developed using the Spring Framework. Spring IoC is used to configure and manage application components.

## Technologies Used

- Java 17
- Maven
- Spring Core
- Visual Studio Code

## Steps Performed

1. Created a Maven project named **LibraryManagement**.
2. Added the Spring Core dependency in `pom.xml`.
3. Created `applicationContext.xml` in the `src/main/resources` directory.
4. Configured Spring beans for `BookService` and `BookRepository`.
5. Implemented the service and repository classes.
6. Created `LibraryManagementApplication` to load the Spring Application Context.
7. Successfully compiled and executed the application.

## Output

The Spring Application Context was loaded successfully, and the application executed without errors.

## Result

Successfully configured a basic Spring application using Spring Core with XML-based configuration.
