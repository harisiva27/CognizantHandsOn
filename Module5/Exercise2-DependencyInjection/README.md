# Exercise 2 - Implementing Dependency Injection

## Objective

Implement Dependency Injection using Spring IoC and XML-based configuration.

## Scenario

A Library Management application uses Spring IoC to inject the `BookRepository` dependency into the `BookService` class using setter injection.

## Technologies Used

- Java 17
- Maven
- Spring Core
- Spring IoC
- XML Configuration
- Visual Studio Code

## Steps Performed

1. Updated `applicationContext.xml` to configure dependency injection.
2. Added a setter method in `BookService` for `BookRepository`.
3. Configured the `BookRepository` bean.
4. Loaded the Spring Application Context.
5. Verified that dependency injection worked successfully.

## Output

The application executed successfully, demonstrating Spring Dependency Injection.

## Result

Successfully implemented Dependency Injection using Spring IoC and XML configuration.
