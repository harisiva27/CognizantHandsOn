# Module 5 - Spring Core

## Overview

This module introduces the fundamentals of the Spring Framework, including Spring Core, Dependency Injection (IoC), XML-based configuration, Spring AOP, and Maven project configuration. The hands-on exercises demonstrate how to build and configure a simple Library Management application using Spring.
