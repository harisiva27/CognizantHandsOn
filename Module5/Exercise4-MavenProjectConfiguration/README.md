# Exercise 4 - Implementing Logging with Spring AOP

## Objective

Implement logging using Spring AOP to monitor method execution within the Library Management application.

## Scenario

The Library Management application requires logging functionality to track the execution of service methods. Spring AOP is used to separate logging functionality from the business logic.

## Technologies Used

- Java 17
- Maven
- Spring Core
- Spring AOP
- AspectJ
- Visual Studio Code

## Steps Performed

1. Added the Spring AOP dependency to the Maven project.
2. Created the `LoggingAspect` class.
3. Configured AspectJ support in `applicationContext.xml`.
4. Registered the logging aspect as a Spring bean.
5. Executed the application to verify AOP-based logging.

## Output

The application executed successfully and displayed log messages before and after method execution.

## Result

Successfully implemented logging using Spring AOP and Aspect-Oriented Programming concepts.
