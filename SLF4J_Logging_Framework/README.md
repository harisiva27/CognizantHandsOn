# Exercise 1 - Logging Error Messages and Warning Levels

## Objective

Learn how to use the SLF4J (Simple Logging Facade for Java) framework to log error and warning messages in a Java application.

## Scenario

A Java application uses the SLF4J logging framework with Logback to record error and warning messages. Logging helps developers monitor application behavior and simplify debugging.

## Concepts Covered

- SLF4J Logging Framework
- Logger
- LoggerFactory
- Error Logging
- Warning Logging
- Logback Integration

## Technologies Used

- Java 17
- Maven
- SLF4J 1.7.30
- Logback 1.2.3
- Visual Studio Code

## Files

- LoggingExample.java

## Expected Output

```text
ERROR com.example.LoggingExample - This is an error message
WARN  com.example.LoggingExample - This is a warning message
```

## Result

✔ Successfully implemented logging using the SLF4J framework to generate error and warning messages.
