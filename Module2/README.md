# Module-2

## Data Structures and Algorithms

This module contains Java implementations of Data Structures and Algorithms hands-on exercises.

### Exercises

#### Exercise-2: Ecommerce Platform Search Function
- Implemented Linear Search and Binary Search algorithms.
- Compared search techniques for finding products efficiently.
- Demonstrated searching a product by its Product ID.

#### Exercise-7: Financial Forecasting
- Implemented recursive forecasting of future financial value.
- Calculated future investment value using a fixed annual growth rate.
- Demonstrated the use of recursion in financial computations.

### Technologies Used
- Java
- Visual Studio Code
- JDK 17

### Learning Outcomes
- Understand Linear Search and Binary Search.
- Analyze search algorithm efficiency.
- Apply recursion to solve financial forecasting problems.
- Improve problem-solving skills using Java.
