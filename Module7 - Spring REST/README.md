# Module 7 - Spring REST using Spring Boot 3

## Exercise 1: Create a Spring Web Project using Maven

### Objective

The objective of this hands-on is to create a Spring Boot web application using Maven and understand the basic structure of a Spring Boot project. It also demonstrates how to configure logging and run the application using the embedded Tomcat server.

### Technologies Used

- Java 17
- Spring Boot 3
- Spring Web
- Spring Boot DevTools
- Maven
- SLF4J Logging

### Features Implemented

- Created a Spring Boot project using Spring Initializr.
- Configured the project with Maven.
- Used the `@SpringBootApplication` annotation.
- Started the application using `SpringApplication.run()`.
- Configured application logging using SLF4J.
- Successfully launched the embedded Tomcat server.

### Output

The application successfully:

- Started the Spring Boot application.
- Initialized the embedded Tomcat server.
- Displayed application startup logs.
- Confirmed successful application startup through the console.

### Learning Outcome

After completing this hands-on, I learned:

- How to create a Spring Boot project using Spring Initializr.
- The purpose of the `@SpringBootApplication` annotation.
- How Spring Boot simplifies Java application development.
- How to configure and use SLF4J logging.
- How the embedded Tomcat server runs a Spring Boot application.
