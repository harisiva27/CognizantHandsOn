
# Week 1 - Design Patterns

This repository contains the mandatory hands-on exercises that I completed during Week 1 of the Cognizant Java FSE Deep Skilling Program.

## Hands-On Completed

### Singleton Pattern
In this exercise, I implemented the Singleton Design Pattern to ensure that only one object of the class is created and shared throughout the application.

### Factory Method Pattern
In this exercise, I implemented the Factory Method Design Pattern to create objects through a factory class instead of creating them directly. This helps make the code more flexible and easier to maintain.

## Concepts Covered

- SOLID Principles
- Singleton Pattern
- Factory Method Pattern
- Object-Oriented Programming (OOP)
